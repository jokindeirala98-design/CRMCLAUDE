En concreto, nuestra idea es:

1. **Comparativa de factura** dentro de Voltis CRM, en la que el comercial sube la factura del cliente o introduce los datos a mano, y obtiene en el momento la propuesta con vuestras tres tarifas 2.0TD (Precio de Mercado, 24 horas y Tramos Horarios), incluyendo ahorro estimado a 6 y 12 meses, igual que el comparador del portal de colaboradores.

Para construir esto correctamente y dentro de los términos de uso adecuados, agradeceríamos mucho que nos confirmaseis los siguientes puntos:

- **¿Existe una API oficial para colaboradores** que dé acceso a (a) cálculo de comparativas y (b) creación de contratos? Si la hay, ¿cómo se solicita el acceso (credenciales, sandbox, manual de integración)?
- **Si no hay API oficial,** ¿cuál es la vía recomendada para enviaros las altas desde un sistema externo (email estructurado, formulario, SFTP, otro)?
- **¿Existen restricciones técnicas o contractuales** sobre el uso del comparador del portal `colaboradores.ganaenergia.com` desde sistemas internos de un colaborador (por ejemplo, automatizar consultas con la sesión del colaborador)?
- **Manuales o documentación técnica** de la integración, si los tenéis disponibles.
- **Persona de contacto técnico** a quien podamos escribir para resolver dudas durante la implementación.
